---
description: Realiza o deploy do projeto para o GitHub e Vercel.
---

Este workflow automatiza o envio de alterações. Lembre-se que o envio para a branch `master` aciona o deploy na Vercel automaticamente.

// turbo
1. Executar o comando de deploy:
```powershell
npm run deploy
```

2. Confirmar o status no GitHub e informar o usuário.
